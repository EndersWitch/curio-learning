📚 Learning Zones
Manage the concept explanations shown to learners before each quiz level
What is a Learning Zone? Before a learner starts a quiz level, they see a screen with concept cards, examples and a list of what will be tested. You manage that content here — one record per level (e.g. one for Nouns I, one for Nouns II, etc). Each record maps to a level_id which must match the ID used in the quiz.

✦ Generate Papers with AI
Bulk-generate CAPS-aligned exam papers using Claude — papers + memos are created, uploaded and saved automatically
How it works: Claude generates a full CAPS-aligned paper (Section A: Comprehension, Section B: Language, Section C: Writing) plus a complete marking memo. Both are rendered as PDFs via the existing PDF engine, uploaded to Supabase Storage (Papers/gr4-english-hl/), and inserted into your papers table — ready to appear on the Papers page immediately.

Tip: Each generation call takes ~30–60 seconds. Start with 1 paper to verify output before generating in bulk.
